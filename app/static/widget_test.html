🎙 AMO Voice Widget — Тестовая страница

Только для разработки. Эта страница не отображается в production-окружении.

Замените токен ниже на реальный widget_token из POST /v1/widgets:

<script src="/widget.js" data-widget-token="wgt_REPLACE_ME"></script>

Виджет появится в правом нижнем углу. Нажмите на кнопку с микрофоном для начала голосовой сессии.

Шаги тестирования

  1. Создать агента в POST /v1/agents
  2. Создать виджет в POST /v1/widgets с agent_profile_id
  3. Скопировать widget_token из ответа
  4. Вставить токен в атрибут data-widget-token скрипта ниже
  5. Перезагрузить страницу
